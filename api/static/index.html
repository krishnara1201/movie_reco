Two-Tower Movie Recommender

Pick movies the user likes, then generate recommendations and inspect why items were selected.

How The Model Predicts

1) Encode liked movies
Each selected movie ID goes through the item tower to get dense vectors.
2) Build user profile
Vectors are averaged then L2-normalized to create a temporary user embedding.
3) Score all movies
Compute dot product: score = user_vec . item_vec for each catalog item.
4) Rank + explain
Top-K by score; contribution chart uses similarity between recommended item and each liked movie.
